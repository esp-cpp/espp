Dev Boards (BSPs)
*****************

Board-support packages (BSPs) for supported development boards, grouped by
vendor. Each BSP wires up the board's display, touch, IMU, audio, buttons, and
other on-board peripherals behind a single, easy-to-use component so you can
target a board without configuring each peripheral by hand.

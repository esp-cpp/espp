Gaussian
********

The `gaussian` class provides an implementation of the gaussian function:

.. math::

   y(t)= \alpha e^{ -\frac{ (t-\beta)^2 }{ 2\gamma^2 } }

The class allows for dynamically changing the :math:`\alpha`, :math:`\beta`, and
:math:`\gamma` parameters.

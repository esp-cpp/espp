Biquad Filter
*************

The `BiquadFilter` class provides an implementation of a `Digital Biequad Filter
<https://en.wikipedia.org/wiki/Digital_biquad_filter>`_ with implementations for
both the `Direct Form 1` and `Direct Form 2`.

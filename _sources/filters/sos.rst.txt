Second Order Sections (SoS) Filter
**********************************

The `SosFilter` class provides an implementation of a Second Order Sections
(SoS) filter. For more information please see `series second-order sections
<https://www.dsprelated.com/freebooks/filters/Series_Second_Order_Sections.html>`_
as well as `Digital Biquad Filter
<https://en.wikipedia.org/wiki/Digital_biquad_filter>`_.

Waveshare ESP32-S3-LCD-1.47 Example

This example shows how to use the espp::WsS3Lcd147 hardware abstraction component initialize the components on the Waveshare ESP32-S3-LCD-1.47.

It initializes the display, button, and RGB LED subsystems.

How to use example

Hardware Required

This example is designed to run on the Waveshare ESP32-S3-LCD-1.47 dev board which has:

  • WiFi / BLE

  • uSD card

  • Color TFT LCD (ST7789, 172x320 1.47” IPS LCD)

  • Boot Button

  • RGB LED

Build and Flash

Build the project and flash it to the board, then run monitor tool to view serial output:

idf.py -p PORT flash monitor

(Replace PORT with the name of the serial port to use.)

(To exit the serial monitor, type Ctrl-].)

See the Getting Started Guide for full steps to configure and use ESP-IDF to build projects.
